# Personal-Finance-Assistance-AI-Agent

Personal Finance Assistance is an AI Agent that will help user in tracking there expenses and help in general queries related to finance using reasoning model with custom tools.


A monorepo containing the server (Node.js) and client (React) applications.

## Getting Started
```bash
# Install dependencies
npm install

# Run both apps in development
npm run dev

# Build all apps
npm run build
```

## Project Structure

- `apps/server` - Node.js backend API
- `apps/llm-client` - React frontend
- `packages/shared` - Shared code between apps

## Commands

- `npm run dev` - Start development servers
- `npm run build` - Build all apps
- `npm run lint` - Lint all apps
- `npm run test` - Run tests
